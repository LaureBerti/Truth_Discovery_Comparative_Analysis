LTM

LTM has a set of parameters, first (K,B and T) for (number Of Iterations, Burn-in and Thin) respectively, a set of values for (K, B, T) was proposed by the original paper authors, we present 2 of them in the following discussion.
As LTM depends on randomization, LTM results for the same dataset is different from one run to another. Thus, the author recommended to compute the precision as the average of the precision of 10 runs of the algorithm. The next results are the average (± standard deviation) of 100 runs, we augmented the number of runs in order to achieve the most accurate results.
