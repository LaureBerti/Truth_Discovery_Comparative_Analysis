In our Parameterization Study, we study the effect of changing the value of each truth discovery algorithm parameter on the algorithm quality.

The study is based on the Book data set, which we reformated in the various formats required by the different truth discovery models:
  • For LTM, the data set should contain only one attribute with a single atomic value, and more than one claimed value can be considered true by LTM.
  • For MLE, the data set should only have Boolean positive value.

So we reformatted the Book data from (ClaimID, SourceID, ThisBook:AuthorNames, (AuthorA,AuthorB)) to
  • (ClaimID1, SourceID, ThisBook:AuthorNames, AuthorA) and (ClaimID2, SourceID, ThisBook:AuthorNames, AuthorB) for LTM and
  • (ClaimID1, SourceID, ThisBook:AuthorA-IsAuthor, True) and (ClaimID2, SourceID, ThisBook:AuthorB-IsAuthor, True) for MLE
