Scalability Experiment

The goal of the scalability experiment is to evaluate the runtime of truth discovery models on large data sets, either varying the number of sources, or the number of objects (and both). We generated 10 synthetic data sets for each case: varying the number of sources from 1,000 and 10,000 sources, and the number of data items from 100, 1,000 to 10,000 data items. Dependence computation based models exceed the memory consumtion limits, LCA and MLE models faced a computational problem when the number of sources is gretaer than 5000. Please refer to the paper for details.
